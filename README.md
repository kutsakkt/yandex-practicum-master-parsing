# yandex-practicum-master-parsing
Итоговый проект по Поиску и сбору данных в совместной магистратуре ТГУ и Яндекс Практикума. Использовался Selenium и Beautiful Soup.

**Задание**: собрать данные о ноутбуках с сайта https://www.nbcomputers.ru/catalog/noutbuki/

Данные включают: Код товара, Ссылка на товар, Название ноутбука, Разрешение экрана, Дискретная видеокарта, Объем оперативной памяти, Тип матрицы, Процессор, Актуальная цена.
